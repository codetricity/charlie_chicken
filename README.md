# Charlie Chicken

![screenshot](assets/docs/screenshot.gif)

Basic sprite animation on Flutter with Flame.

Uses Flame joystick.

Add custom maps with Tiled and load into game with `flame_tiled`.

![custom map](assets/docs/custom_map.gif)
